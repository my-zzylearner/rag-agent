# 项目效果检测与评估

## 核心定义
项目评估是对项目全生命周期的系统性分析，旨在验证目标达成度、过程合规性及最终商业价值。

## 评估方法
- **过程管理**：监控执行阶段关键节点，识别偏差并及时纠偏。
- **质量评价**：依据交付物标准进行验收，确保符合预期。
- **价值分析**：通过 ROI 等模型量化项目对业务的贡献。

## 关键指标 (KPI)
1. **进度指标**：计划完成率、里程碑达成率。
2. **成本指标**：预算偏差率、实际支出。
3. **质量指标**：缺陷密度、客户满意度。
4. **效益指标**：投资回报率、资源利用率。

## 实施建议
需结合定量数据与定性反馈，建立常态化监测机制，动态调整策略以确保项目成功。
