# 阿里妈妈淘宝联盟

*   **定义**
    隶属于阿里妈妈旗下的全网推广营销平台，旨在连接商家（广告主）与推广者（流量主）。
*   **核心原理**
    采用 CPS（Cost Per Sale，按成交付费）模式。商家设置商品佣金比例，推广者通过各类渠道（如网站、APP、社交媒体）推广商品链接。用户购买成交后，推广者获得佣金，平台提供技术支持与结算服务。
*   **主要角色**
    *   **商家**：提供商品与佣金，获取销售流量。
    *   **推广者**：负责推广引流，赚取推广收益。
    *   **平台**：提供数据追踪、结算及生态管理。
