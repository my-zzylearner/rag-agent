### Q4: 什么是 LLM Agent？

**标准答案：** LLM Agent 是以大语言模型为核心推理引擎，能够自主规划、调用外部工具、循环执行直到完成任务的系统。

**RAG 回答：** 根据参考内容，LLM Agent 是以大语言模型为核心推理引擎，能够自主规划、调用外部工具、循环执行直到完成任务的系统。

其主要特点如下：
1.  **基本原理与特征**：包含感知-决策-执行循环，具备自主性和目标导向。
2.  **技术基础**：涵盖大模型、记忆机制、推理能力、工具使用。
3.  **与普通 LLM 的区别**：
    *   **普通 LLM**：输入 → 输出，一次性生成。
    *   **LLM Agent**：输入 → 思考 → 行动 → 观察 → 思考 → ... → 输出，循环迭代。
4.  **核心组件**：
    *   **规划**：负责分解任务、制定步骤、决定调用哪个工具。常见范式包括 ReAct、Chain-of-Thought 和 Tree of Thoughts。
    *   **工具**：Agent 可以调用的外部能力（如搜索引擎、代码执行器、数据库查询、API 调用、文件读写），用于扩展 LLM 的边界。
